{void} enableToolbarHelp(enabled)
Specifies whether the tool bar help feature must be enabled. This code initializes a new viewer display, using enableToolbarHelp(true) to enable the toolbar help feature: 
	function initDisplay( ){
		var uioptions = new actuate.viewer.UIOptions( );
		viewer = new actuate.Viewer("viewerpane");
		var viewerwidth = 800;
		var viewerheight = 600;
		viewer.setWidth(viewerwidth);
		viewer.setHeight(viewerheight);
		uioptions.enableToolBar(true);
		uioptions.enableToolbarHelp(true);
		viewer.setUIOptions(uioptions);
		document.getElementById("display").disabled = false;
	}
Parameters:
{boolean} enabled
true for enabled, false for disabled.
Returns:
{void}
